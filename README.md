# Task 1: Customer Behavior Analysis

---

## Workflow

### 1. Data Collection
* Imported customer transaction data containing demographics, purchases, payment methods, returns, and churn information.

### 2. Data Cleaning
* Handled missing values, removed inconsistencies, validated data types, and ensured data quality for analysis.

### 3. Feature Engineering
* Created new analytical features such as:
  * Age Groups
  * Spending Segments
  * Purchase Frequency
  * **RFM Metrics** (Recency, Frequency, Monetary)
  * Customer Segments

### 4. Exploratory Data Analysis (EDA)
* Analyzed customer behavior patterns across:
  * Revenue Trends
  * Demographics
  * Product Categories
  * Payment Methods
  * Return Status

### 5. Customer Segmentation
* Applied **RFM Analysis** to classify customers into:
  * Champions
  * Loyal Customers
  * Potential Loyalists
  * At Risk
  * Lost Customers

### 6. Churn Analysis
* Examined churn patterns across:
  * Gender
  * Age Groups
  * Product Categories
  * Payment Methods
  * Customer Segments

### 7. Business Insights
* Identified key revenue drivers, customer retention trends, and customer lifetime value opportunities.

### 8. Recommendations
* Provided actionable strategies to improve:
  * Customer Engagement
  * Retention
  * Revenue Growth

### 9. Dashboard Development
> **Final Deliverable:** Built an interactive **Power BI dashboard** to visualize KPIs, customer segments, churn metrics, and revenue trends.
